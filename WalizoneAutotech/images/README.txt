This directory contains images for the Walizone Autotech website.

Recommended image structure:
- hero-bg.jpg: Hero section background image (1920x1080px recommended)
- logo.png: Company logo (transparent background)
- product-placeholder.jpg: Default image for products without photos
- vehicle-placeholder.jpg: Default image for vehicle listings without photos
- services/: Folder for service-related images
- products/: Folder for product images
- dealership/: Folder for dealership listing images

Image naming conventions:
- Use lowercase letters and hyphens instead of spaces
- Include descriptive names (e.g., "engine-repair-service.jpg")
- For product images, include the product ID (e.g., "product-123.jpg")
