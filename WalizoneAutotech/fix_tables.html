Fix Database Tables

Important: This page will help you fix database issues with the vehicle management system. Please read the instructions carefully before proceeding.

Step 1: Fix Database Tables

Click the button below to run the database fix script. This will:

  • Add missing columns to the vehicles table
  • Create the service_history table if it doesn't exist
  • Fix any data type issues in the code
Run Fix Script

Step 2: Return to My Vehicles

After running the fix script, you can return to the My Vehicles page to check if the issues have been resolved.

Go to My Vehicles

Alternative: Manual SQL Fix

If the automatic fix doesn't work, you can manually run the SQL commands in phpMyAdmin:

  1. Open phpMyAdmin
  2. Select your database (walizone_autotech)
  3. Go to the SQL tab
  4. Copy and paste the contents of the fix_database.sql file
  5. Click "Go" to execute the SQL commands
